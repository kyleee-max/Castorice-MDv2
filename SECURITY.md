# 🔐 Security Overview

Castorice MD dibuat dengan sistem modular yang memungkinkan pengguna
untuk:
- memodifikasi source code
- menambahkan plugin
- menghapus fitur
- mengembangkan sistem sendiri

Karena sifat project yang terbuka dan dapat dikembangkan bebas,
pengguna bertanggung jawab penuh atas seluruh perubahan yang dilakukan
pada Software.

# ⚠️ Important Notice

FREE VERSION tidak menggunakan:
- License Key
- API License Validation
- Remote Access System
- Hidden Backdoor
- Remote Kill Switch
- Forced Verification

Seluruh source code dapat diperiksa dan dikembangkan secara mandiri.

---

# 🛡️ Security Recommendation

Untuk menjaga keamanan bot, pengguna disarankan untuk:

- Tidak menjalankan plugin dari sumber tidak terpercaya
- Tidak memberikan akses owner kepada sembarang pengguna
- Tidak menyimpan data sensitif di plugin publik
- Tidak menjalankan command eval/exec sembarangan
- Menggunakan server/VPS yang aman
- Menggunakan Node.js versi terbaru yang stabil

---

# 🚫 Prohibited Activities

Aktivitas berikut tidak diperbolehkan:

- Menjual script dalam bentuk apa pun
- Menghapus copyright atau credit author
- Mengganti identitas kepemilikan project
- Mengklaim project sebagai milik pribadi
- Menyebarkan versi crack, nulled, atau bypass

Pelanggaran terhadap ketentuan tersebut dapat dikenakan tindakan sesuai
Lisensi yang berlaku.

disebabkan oleh modifikasi pihak ketiga.
